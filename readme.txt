== Description ==

A child theme for Understrap (Underscores and Bootstrap). Ingeni Starter is a simple starting point for Gutenberg-based theme development.


Website: https://ingeni.net

Child Theme Project: https://github.com/BruceMcKinnon/ingeni-starter.git

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Understrap in the search form and press the 'Enter' key on your keyboard.
3. Unzip ingeni-starter into your Wordpress themes folder.
3. Modify the ingeni-starter theme files as required.
